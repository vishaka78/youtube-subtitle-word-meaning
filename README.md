# YouTube Subtitle Word Meaning Extension

A Chrome extension that shows word meanings when users click on YouTube subtitles.

## Features
- Click any subtitle word to get its meaning
- Works directly on YouTube captions
- Lightweight and fast
- Helps in language learning

## Technologies Used
- JavaScript
- Chrome Extension API
- Dictionary API

## How to Use
1. Install the extension in Chrome
2. Open YouTube
3. Turn ON subtitles
4. Click any word in subtitles to get its meaning
